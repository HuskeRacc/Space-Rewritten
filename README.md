# Space-Rewritten
Space Game Premise

Player lost in space. No fuel. The player must use a drone to gain materials which can be used to craft food, generator fuel and thruster fuel.
The player’s goal is to get enough thruster fuel to move on to a station.

The drone can mine (3) materials to start:
•	Satonium
•	Thrustium
•	Fuelium
Satonium can be turned into food products.
Thrustium is the rare and needed material to create fuel for the thrusters.
Fuelium is used to create generator fuel to keep the oxygen generator and lights running.

The drone randomly collects these materials at a random rate. 
The player can set a collection mode via the Helm screen menu and manipulate how quickly the drone mines certain materials.
The drone modes for now consist of:
•	Everything mode (Mode 0)
•	Satonium Priority (Mode 1)
•	Fuelium Priority (Mode 2)

The player must manage (3) needs:
•	Hunger
•	Fatigue
•	Oxygen
Hunger is amended by crafting food via Satonium.
Fatigue is amended by sleeping.
Oxygen is amended by keeping the oxygen generator running.
