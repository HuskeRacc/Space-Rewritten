# Space Rewritten

Space Rewritten is a Unity space survival prototype about being stranded in deep space with limited fuel, oxygen, and resources.

The player must survive aboard a damaged ship by managing hunger, fatigue, oxygen, generator fuel, and ship systems. A mining drone can be sent out to collect materials, which are used to craft food, generator fuel, upgrades, and eventually thruster fuel. The long-term goal is to produce enough thruster fuel to reach a nearby station.

The game currently features drone resource collection, material banking, crafting, ship status screens, generator refuelling, oxygen management, drone damage, upgrades, and random ship system failures.
