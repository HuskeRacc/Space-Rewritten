# Security Policy

## Reporting a Vulnerability

This is a personal Unity prototype and is not currently accepting formal security reports.

If you notice a serious issue, exposed secret, or licensing/security concern, please contact the repository owner directly through GitHub.

Please do not publicly disclose security issues before they have been reviewed.

## Supported Versions

Only the latest version on the `main` branch is considered active.
Older releases and `indev-*` builds are archived and unsupported.
